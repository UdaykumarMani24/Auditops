# AuditOps Scientific Demo

A scientific demonstration of the AuditOps framework for continuous compliance in ML systems.

## Overview
This repository contains a scientific validation of the AuditOps framework using real medical data from the UCI Cleveland Heart Disease dataset.

## Features
- Real medical dataset analysis (UCI Cleveland Heart Disease)
- Cross-validation with statistical significance testing
- Regulatory compliance validation (FDA Class II)
- Automated audit trail generation
- Explainability analysis (SHAP, feature importance)
- Compliance metrics calculation (CCS, EPI, ATCM, RDDR)
